## 🖼️ Меню обоев

Меню для выбора статичных или живых (видео) обоев с превью.

### Возможности
- Просмотр статичных обоев (`jpg`, `png`, `jpeg`) с превью
- Просмотр живых обоев (`mp4`, `webm`, `gif`) с автоматически сгенерированными превью
- Превью кешируются в `/tmp/wallpaper-previews` для быстрого повторного запуска
- Автоопределение языка (русский / английский) на основе локали системы
- При `Esc` возврат в главное меню, после выбора обоев меню закрывается

### Установка

1. Скопируй файлы в нужные директории:
```bash
cp wallpaper.sh ~/.config/rofi/wallpaper.sh
cp wallpaper.rasi ~/.config/rofi/wallpaper.rasi
chmod +x ~/.config/rofi/wallpaper.sh
```

2. Создай папки для обоев:
```bash
mkdir -p ~/Wallpaper/live
```

3. Положи статичные обои в `~/Wallpaper/`, живые в `~/Wallpaper/live/`
4. Скачай шрифт или замени на свой в файле *.rasi (2 строка)
### Запуск
```bash
~/.config/rofi/wallpaper.sh
```

### Управление
| Клавиша | Действие |
|---|---|
| `Enter` | Выбрать обои |
| `Esc` | Вернуться в главное меню |

### Структура папок
```
~/Wallpaper/          # Статичные обои
~/Wallpaper/live/     # Живые обои (mp4, webm, gif)
```

### Как это работает
1. Открывается главное меню с вариантами: Живые / Статичные / Выход
2. После выбора категории появляется список обоев с миниатюрами
3. Для живых обоев — ffmpeg генерирует превью из первого кадра видео
4. После выбора — xwinwrap + mpv рендерит видео как фон рабочего стола
5. Для статичных обоев — feh устанавливает фон

## ⚡ Меню питания

Простое меню для управления питанием системы.

### Возможности
- Выключение, перезагрузка, сон
- Автоопределение языка (русский / английский)
- Минималистичный дизайн без поля ввода текста

### Установка

```bash
cp power.sh ~/.config/rofi/power.sh
cp power.rasi ~/.config/rofi/power.rasi
chmod +x ~/.config/rofi/power.sh
```

### Запуск
```bash
~/.config/rofi/power.sh
```

### Управление
| Клавиша | Действие |
|---|---|
| `Enter` | Подтвердить действие |
| `Esc` | Закрыть меню |

### Действия
| Пункт | Команда |
|---|---|
| Выключить | `poweroff` |
| Перезагрузить | `reboot` |
| Сон | `suspend` |

## 🎨 Тема оформления

Оба меню используют розово-тёмную цветовую схему. Цвета можете поменять сами в *.rasi конфигах, там всё интуитивно понятно

### Цветовая палитра
| Назначение | Hex | Описание |
|---|---|---|
| Фон | `#1e1020` | Фон окна |
| Поверхность | `#2e1530` | Фон строки ввода |
| Акцент | `#f065a0` | Рамка, выделение |
| Текст | `#f0a0c0` | Основной текст |
| Текст выбранного | `#1e1020` | Текст на выбранном элементе |
